# Texas Food Banks Directory

A responsive website that provides information about food banks across major cities in Texas, including Dallas, Houston, and Austin. The goal of this project is to help users easily find food assistance resources, view contact details, and get directions to nearby food banks.

---

## ğŸŒ Project Overview

Food insecurity affects millions of Texans each year. This website serves as a centralized directory where users can:

- Learn about food insecurity in Texas
- Browse food banks by city
- View food bank details such as address, phone number, services offered, and website
- Get directions to food bank locations
- Submit a request to add a new food bank listing

The project is built using HTML and CSS, with a focus on semantic structure, Flexbox layouts, and responsive design.
